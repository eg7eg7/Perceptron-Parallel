בתיקיית פרויקט סיום נמצאים שני קבצי קלט לדוגמה. בשניהם הוגדרו נקודות במישור דו ממדי.

באותה תיקיה נמצאת תוכנית עזר גראפית BinaryClassification.jar שבעזרתה אפשר להציג את הנקודות מהקובץ על המסך. הנקודות עם הסימן 1 הן בצבע כחול והנקודות עם הסימן -1  הן בצבע אדום. כדי להציג את הנקודות יש להקליד

java -jar BinaryClassification.jar data1.txt

אפשר להעביר את הפרמטרים a, b, c כדי לצייר גם את הישר  ax + by +c = 0

java -jar BinaryClassification.jar data1.txt   a  b  c
